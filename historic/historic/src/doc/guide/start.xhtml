This guide provides an introduction to JAPI.

Contents

Please choose the desired JAPI subject from the available guides:

IO Guide
Usage of ARGV classes to write programs that use a behaviour comparable with Perl's <ARGV>.
Logging Guide
Usage of JAPI's Logging API.
Action Guide
Usage of ActionBuilder and related classes like DummyAction, ReflectionAction and ToggleAction for easily handling actions with localization and preferences support.
TipOfTheDay Guide
Usage of TipOfTheDayManager for adding Tip Of The Day dialogs to your application.
XML Guide
Usage of tiny little extra XML classes that make your life with XML in Java easier.
