JAPI Library Taglets

What is JAPI Library Taglets?

JAPI Library Taglets is a collection of taglets that are used by the JAPI developers during JAPI development. They might also be useful for your Java project if you use Javadoc.

Taglet Index (alphabetical)

  • @fixme

Taglet Index (grouped)

  • Simple Documentation Taglets
    • @note
    • @warning
  • Revisit-Taglets
    • @fixme
    • @todo
    • @xxx
  • Contract Taglets
    • @invariant
    • @pre
    • @post
    • @retval

@todo

The @fixme taglet can be used to denote code that's known to be incomplete and needs completion later.

@warning

The @warning taglet can be used to highlight warnings for users of a symbol.

@xxx

The @xxx taglet can be used to denote code that looks like being broken yet seems to work but should be reviewed and refactored later.
