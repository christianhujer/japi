This package contains classes and interfaces for managing a list of recently opened files.

Usage

  • Alternatively,
    • Create and instanciate your desired implementation of {@link net.sf.japi.swing.recent.RecentURLs}.
    • Get an instance using {@link net.sf.japi.swing.recent.PrefsRecentURLs#PrefsRecentURLs(Class)}
  • Somewhere implement {@link net.sf.japi.swing.app.CanLoad} so a recently loaded document can be activated.
  • Create a Menu for recently opened files (URLs) using {@link net.sf.japi.swing.recent.RecentURLsMenu#RecentURLsMenu(net.sf.japi.swing.recent.RecentURLs,net.sf.japi.swing.app.CanLoad)}
