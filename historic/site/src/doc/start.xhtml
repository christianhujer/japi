What is JAPI?

JAPI is a set of Java API extension libraries intended to make life easier for Java application developers. Currently it is targetted on normal Desktop Java (J2SE). JAPI helps you writing internationalized and localized Swing applications with very little effort and very efficient event handling code. Additionally it contains some useful classes about argument parsing, I/O and XML. Note: JAPI requires J2SE 5.0 or later.
